Support

Need help with SubmitPilot? Send a concise support request and include enough context to reproduce the issue.

Contact

Email support@freerangelabs.dev.

Please include your macOS version, SubmitPilot version, whether the issue happens with a new target app audit or an existing audit, and the steps that led to the problem.

Common Topics

  • Creating or resetting a target app readiness audit.
  • Using risk tags to add conditional checklist items.
  • Adding screenshots, links, files, and notes to evidence.
  • Exporting a Markdown readiness report.
  • Copying App Review notes for App Store Connect.

What Not To Send

Do not send signing certificates, provisioning profiles, passwords, private source code, unreleased customer data, or App Store Connect credentials.
