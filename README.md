# SubmitPilot

App review readiness for every target app.

SubmitPilot is a native macOS checklist and evidence organizer for developers preparing apps for App Store submission. It helps teams review common launch-readiness areas such as metadata, privacy links, permissions, subscriptions, account deletion, review notes, and release evidence for each target app.
